Attach a hen to start.

Each hen runs in a persistent tmux session. If the hen was raised with an agent kind, that CLI launches automatically on first connect — otherwise type codex, claude, or copilot yourself.

Sessions survive page reloads and reconnects. Ctrl-b d to detach · Ctrl-b c for a new window · Ctrl-b n/p to cycle windows.

📋 Farm-wide tasks

Submit a task — the first idle hen with a matching agent kind picks it up. Use Any CLI agent to let the queue pick whichever (claude / codex / gh-copilot) is available.

🛒 Cross-coop compute market

List spare hen capacity to neighbouring coops, or rent compute from them. Bid & ask are exchanged over a 256-byte gossip frame on the World relay; settlement is sealed-bid escrow in Grain.

The market layer ships in a separate repository, coop-market, which keeps the proprietary listing / escrow / federation code out of the OSS daemon (see open-core split in AGENTS.md).
